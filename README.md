# ReplayPilot

Step 1 scaffolds a monorepo for the UI Navigator project.

## Prerequisites

- Node.js 20

## Install

```bash
npm install
```

For local Playwright screenshots, also install Chromium:

```bash
npx playwright install chromium
```

## Run in development

```bash
npm run dev
```

## URLs

- Controller: http://localhost:5173
- API health: http://localhost:8080/health

Step 1 only includes the API health endpoint.
